# 🤖 AI-Powered ATS Resume Screening Bot

Automated recruitment pipeline — zero manual effort.

## Tech Stack
- n8n (workflow automation)
- Google Gemini AI
- Gmail API

## How it works
1. Candidate submits form with PDF resume
2. n8n parses resume automatically
3. Gemini AI scores resume & gives hire/reject recommendation
4. Automated email sent to HR and candidate
